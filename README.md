# Usage

Downloaden, run.bat ausführen und warten.

# First Run
beim starten wird geprüft, ob alles was benötigt wird installiert ist. sollte das nicht der fall sein, startet run.bat automatisch den python installer.


# Motility ans startmenu heften
zuerst den ordner an einen Ort packen, wo er immer bleiben kann. Dann eine Verknüpfung erstellen, die auf die run.bat zeigt. Dann WIN + R drücken, "%appdata%\Microsoft\Windows\Start Menu\Programs" eingeben, ENTER drücken, und die Verknüpfung in diesen ordner Verschieben

# Epl Zpl Converter
Für den Epl Zpl Converter muss Image Magic installiert sein.
windows install command: 
```console
winget install ImageMagick.Q16
```
